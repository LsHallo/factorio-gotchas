# Get the Latest Beta

How to get access to the latest Factorio beta release on Steam or through the official website

# Steam

Right click Factorio and select "Properties"
Go to the "Betas" tab in the top right
Select latest experimental and close the window
Wait for the steam update to finish and enjoy your play on the bleeding edge!

# Website

Log in on factorio.com
Go to the download section in the game dropdown
Download the latest experimental for your platform
